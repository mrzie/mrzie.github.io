# StyledEditor 快捷键指南 (Typora 兼容)

本文档记录了 StyledEditor 支持的所有快捷键及其对应的 Markdown 语法效果。

> **说明**：`Mod` 在 Mac 上对应 `Command (⌘)`，在 Windows/Linux 上对应 `Ctrl`。

## 一、 行内格式 (Format)

| 功能 | Mac 快捷键 | Win/Linux 快捷键 | 效果 |
| :--- | :--- | :--- | :--- |
| **加粗** | `Mod + B` | `Mod + B` | `**文本**` |
| **斜体** | `Mod + I` | `Mod + I` | `*文本*` |
| **下划线** | `Mod + U` | `Mod + U` | `<u>文本</u>` |
| **删除线** | `Mod + Shift + 5` | `Alt + Shift + 5` | `~~文本~~` |
| **行内代码** | `Mod + Shift + \`` | `Mod + Shift + \`` | `` `代码` `` |
| **插入超链接** | `Mod + K` | `Mod + K` | `[描述](url)` |
| **插入图片** | `Mod + Control + I` | `Mod + Shift + I` | `![alt](url)` |
| **清除格式** | `Mod + \` | `Mod + \` | 移除选中区域的行内标记 |

## 二、 段落样式 (Paragraph)

| 功能 | Mac 快捷键 | Win/Linux 快捷键 | 效果 |
| :--- | :--- | :--- | :--- |
| **标题 1-6** | `Mod + 1~6` | `Mod + 1~6` | `#` 到 `######` |
| **普通正文** | `Mod + 0` | `Mod + 0` | 移除当前行的块级前缀 |
| **提升标题级别** | `Mod + =` | `Mod + =` | 增加一个 `#` |
| **降低标题级别** | `Mod + -` | `Mod + -` | 减少一个 `#` |
| **引用** | `Mod + Opt + Q` | `Mod + Shift + Q` | 行首插入 `> ` |
| **无序列表** | `Mod + Opt + U` | `Mod + Shift + ]` | 行首插入 `- ` |
| **有序列表** | `Mod + Opt + O` | `Mod + Shift + [` | 行首插入 `1. ` |
| **任务列表** | `Mod + Opt + X` | `Mod + Shift + X` | 行首插入 `- [ ] ` |
| **分割线** | `Mod + Opt + -` | `Alt + Shift + -` | 插入 `---` |

## 三、 插入复杂块 (Insert Block)

| 功能 | Mac 快捷键 | Win/Linux 快捷键 | 说明 |
| :--- | :--- | :--- | :--- |
| **表格** | `Mod + Opt + T` | `Mod + T` | 插入 3x2 表格模板 |
| **代码块** | `Mod + Opt + C` | `Mod + Shift + K` | 插入 ``` 代码围栏 |
| **数学公式块** | `Mod + Opt + B` | `Mod + Shift + M` | 插入 $$ 公式块 |

## 四、 编辑与选择 (Edit & View)

| 功能 | Mac 快捷键 | Win/Linux 快捷键 | 效果 |
| :--- | :--- | :--- | :--- |
| **增加缩进** | `Mod + ]` | `Mod + ]` | 列表缩进或行缩进 |
| **减少缩进** | `Mod + [` | `Mod + [` | 减少缩进 |
| **选中当前词** | `Mod + D` | `Mod + D` | 选中/扩展选中单词 |
| **选中当前行** | `Mod + L` | `Mod + L` | 选中光标所在整行 |
| **源代码模式** | `Mod + /` | `Mod + /` | 切换 Markdown 源码视图 |
